# A GLPI full and timestamped backup Bash script
I wrote a full timestamped GLPI backup Bash script saving the GLPI directory with tar command and the GLPI MariaDB Database with mariadb-dump command.
In this sample :
- /home/www/glpi is the GLPI files directory ;
- glpi is th GLPI MariaDB Database ;
- /home/temp is the backup directory.

Source : https://www.dsfc.net/infra/sauvegarde/sauvegarde-glpi/
